# Suthoju Ganesh — Full Stack Portfolio

A premium, modern portfolio website showcasing full-stack development, applied AI engineering, and cloud architecture projects. 

## 🚀 Live Deployment
Check out the live website deployed on Vercel:
**[Live Demo & Vercel Deployment](https://vercel.com/suthoju-ganeshs-projects/portfolio-ykpa/AVKwpgt6G82y24UvEyWCvpL897qg)**

---

## ✨ Key Features
- **Adaptive Light/Dark Theme**: Smooth transitions between a futuristic dark mesh mode and a clean, high-contrast light frosted glass mode (persisted via `localStorage`).
- **Interactive Visuals**: Beautiful custom canvas particle background, custom cursor, typing text animations, and smooth layout triggers.
- **Projects Hub**: Curated showcase of full-stack, AI, blockchain, and data projects with custom-generated high-fidelity landing page mockups and detail modals.
- **Personal Branding**: Integrated custom Lord Ganesha **SG** brand logo across the header and footer.
- **Form Integration**: Fully functional contact form with client validation.

---

## 🛠️ Tech Stack
- **Framework**: [TanStack Start](https://tanstack.com/router/v1/docs/start/overview) (Server-Side Rendering + Vite + Nitro bundler)
- **Frontend library**: React 19
- **Styling**: Tailwind CSS v4 & Vanilla CSS
- **Animations**: Framer Motion & CSS keyframe floats
- **Icons**: Lucide React
- **Type Safety**: TypeScript

---

## 💻 Local Development

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run dev server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:8080](http://localhost:8080) in your browser.

3. **Build for production**:
   ```bash
   npm run build
   ```
   *Note: Built with Nitro integration (`nitro: true` enabled in `vite.config.ts`), making it fully compatible with Vercel serverless deployments.*
